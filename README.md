# The NERDTerm

The NERDTerm is a terminal toggle function for the Vim editor.

## Installation

Install via your preferred package manager. Example using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'wuelnerdotexe/nerdterm'
```
